Lienzo técnico de transmisión

Elija un área de trabajo abajo: cada pestaña muestra solo ese tipo de elementos en el lienzo (el modelo completo se mantiene al cambiar de pestaña). Clic para seleccionar; doble clic para abrir propiedades. Use la rueda para zoom y las flechas para desplazar la vista.

Engranajes

Mismo módulo y acercamiento → encaje teórico. Use la barra sobre el lienzo para añadir o vaciar. Arrastre para mover.

Poleas y correas

Arrastre las poleas. Con 2+ poleas el trazado (casco convexo / tangentes) envuelve todos los centros. Vista previa discontinua; al Cerrar correa pasa a sólido. «Elegir orden» + clic corto por polea para secuencia manual.

Piñones y cadenas

Con 2+ piñones la vista previa cierra el lazo por geometría exterior (como correas). «Elegir orden cadena» + clic corto define la secuencia.
