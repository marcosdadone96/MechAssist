R Datos de entrada

Modelo orientativo para transportador de rodillos motorizado horizontal. El calculo entrega par, velocidad y potencia para seleccionar motorreductor.

Ayuda para introducir datos
  • Crr: 0.02-0.05 para lineas en buen estado; suba valor si hay suciedad o rodamientos gastados.
  • Resistencia adicional: use este campo para topes, guias laterales, transferencias o acumulacion.
  • Tiempo de aceleracion: tiempos muy cortos aumentan el pico de par.
  • Factor de servicio: se autocompleta por tipo de carga, o puede usar personalizado.
