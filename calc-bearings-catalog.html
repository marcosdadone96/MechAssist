Rodamientos rígidos de bolas · series 6000 / 6200 / 6300 (catálogo demo)

Datos de muestra: series 62xx con C orientativo. Para selección final, consulta catálogo oficial SKF o FAG.

Elija serie y designación con C demo; obtenga L10h con P y n. Para solo validar vida con una C ya conocida, use el módulo L10 genérico.

Contexto ampliado y notas de uso

Con este asistente elige una designación comercial de las series 6000, 6200 o 6300 y obtiene la vida básica en horas L10h usando la carga dinámica C del catálogo de demostración, frente a la carga equivalente y la velocidad de trabajo. El enfoque sigue el planteamiento ISO 281 para rígidos de bolas en versión abreviada, útil para cribar tallas antes de abrir el catálogo completo del fabricante. Interesa a técnicos de mantenimiento y diseño que deben proponer un reemplazo rápido. Un escenario frecuente es comparar un 62xx frente a un 63xx más ancho cuando sube el empuje axial por un mal alineado.

Usa este módulo para seleccionar un rodamiento de las series 6000/6200/6300 directamente por designación. Si ya tienes el valor C del catálogo, usa el módulo L10 genérico.

Metodología y límites del modelo

Elija serie y designación; introduzca P equivalente y n. Se calcula L10h con L = (C/P)³ (bolas) frente a horas objetivo. Catálogo y C son demo; confirme siempre con el catálogo del fabricante.
